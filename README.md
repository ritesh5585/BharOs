# 🌌 BharOS

> A macOS-inspired interactive desktop experience built with React.

BharOS is a frontend-focused desktop UI project designed to simulate a real operating system experience directly inside the browser.

The project focuses on:

* draggable & resizable app windows
* real-time date and clock system
* live battery status detection
* smooth animated dock
* desktop-style interaction flow
* minimal Notes & Files apps
* browser-powered desktop experience

---

# ✨ Features

* 🖥️ Draggable window system using `react-rnd`
* 🕒 Real-time navbar clock & date
* 🔋 Live battery percentage + charging detection
* 🚀 Smooth animated macOS-style dock
* 🔍 Quick Google search launcher
* 📁 Minimal Files app UI
* 📝 Clean Notepad-inspired Notes app
* 🎨 Modern dark desktop interface
* ⚡ Fast Vite-powered frontend

---

# 🛠️ Tech Stack

* React
* Vite
* SCSS
* react-rnd

---

# 🚀 Run Locally

```bash
npm install

```
