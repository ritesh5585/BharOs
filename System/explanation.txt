PART 1: SPOTIFY REAL-TIME INTEGRATION — iframe कैसे काम करता है?
================================================================

━━ iframe क्या होता है? ━━

<iframe> एक HTML element है जो आपके browser के अंदर एक
दूसरी website को embed (घुसाने) का काम करता है।

यह बिल्कुल वैसा है जैसे एक frame में तस्वीर रखते हैं —
फर्क यह है कि इस frame में एक पूरी live website चलती है।

Example:
  आपका BharOS window  →  अंदर iframe  →  open.spotify.com

━━ Spotify का iframe URL कहाँ से आया? ━━

Spotify एक official "Embed" feature देता है।
URL format:
  https://open.spotify.com/embed/playlist/<PLAYLIST_ID>?utm_source=generator

यह URL real-time Spotify player को embed करता है।
इसमें:
  - Song list दिखती है
  - Play/Pause/Skip controls होते हैं
  - अगर user Spotify account से logged in है → real songs चलते हैं
  - बिना login → 30-second preview चलता है

━━ Spotify.jsx का code कैसे काम करता है? ━━

FILE: src/components/Apps/Spotify.jsx

```jsx
const Spotify = ({ onClose }) => {
  return (
    <AppWindow title="Spotify" onClose={onClose}>
      <iframe
        title="Spotify Embed"
        style={{ borderRadius: "12px", display: "block" }}
        src="https://open.spotify.com/embed/playlist/36AbvtVfjUj7kf5jMha1E6?utm_source=generator&theme=0"
        width="100%"
        height="100%"
        frameBorder="0"
        allowFullScreen={true}
        allow="autoplay; clipboard-write; encrypted-media; fullscreen; picture-in-picture"
        loading="lazy"
      />
    </AppWindow>
  );
};
```

LINE-BY-LINE समझाइए:

  { onClose }       → यह prop AppWindow को मिलता है।
                      जब user X (close) दबाए तो window बंद हो।

  <AppWindow>       → BharOS की draggable, resizable window।
                      इसमें titlebar और close/minimize/maximize buttons हैं।

  <iframe src="...">→ यह actual Spotify player load करता है।
                      Browser इस URL को fetch करता है और
                      Spotify की website को एक mini-window में show करता है।

  width="100%"      → iframe window की पूरी चौड़ाई लेता है।
  height="100%"     → iframe window की पूरी ऊंचाई लेता है।

  allowFullScreen   → Spotify को fullscreen mode में जाने देता है।
  allow="autoplay"  → Spotify को automatically music चलाने की permission।

  loading="lazy"    → जब window खुले तभी iframe load हो, पहले नहीं।
                      यह performance के लिए अच्छा है।

━━ React में style कैसे लिखें? ━━

HTML में:   style="border-radius: 12px"       ❌ React में नहीं चलेगा
React में: style={{ borderRadius: "12px" }}   ✅ यह सही तरीका है

कारण: React JSX में style एक JavaScript Object होता है,
       string नहीं। इसलिए double {{ }} लगता है।

━━ allowfullscreen vs allowFullScreen ━━

HTML में:    allowfullscreen=""              ❌ React warning देता है
React में:  allowFullScreen={true}          ✅ React का camelCase format

PART 2: MULTIPLE WINDOWS — एक साथ कई apps कैसे खुलती हैं?
================================================================

━━ पुरानी system (Single Window — Problem) ━━

FILE: App.jsx (पहले ऐसा था)

  const [openApp, setOpenApp] = useState(null);

यह एक single string variable था।
जब आप Terminal खोलते: openApp = "terminal"
जब आप Spotify खोलते:  openApp = "spotify"  ← Terminal बंद!

Problem: एक variable में एक ही value होती है।
         नई app खुलते ही पुरानी app बंद हो जाती थी।

━━ नई system (Multiple Windows — Solution) ━━

FILE: App.jsx (अब ऐसा है)

  const [openApps, setOpenApps] = useState([]);

यह एक ARRAY है। Array में multiple values हो सकती हैं।
जैसे: ["terminal", "spotify", "notes"]

━━ openAppHandler — App खोलने का function ━━

  const openAppHandler = (appName) => {
    setOpenApps((prev) =>
      prev.includes(appName) ? prev : [...prev, appName]
    );
  };

यह function:
  - Check करता है: क्या यह app पहले से खुली है?
  - अगर हाँ  → कुछ नहीं करता (duplicate नहीं बनता)
  - अगर नहीं → उसे array में जोड़ देता है

Example:
  openApps = ["terminal"]
  User Spotify पर click करता है
  openAppHandler("spotify") call होता है
  openApps = ["terminal", "spotify"] ✅ दोनों खुली!

━━ closeAppHandler — App बंद करने का function ━━

  const closeAppHandler = (appName) => {
    setOpenApps((prev) => prev.filter((app) => app !== appName));
  };

  .filter() → सिर्फ वो items रखता है जो condition pass करें।
              जो app close होनी है, वो condition fail करती है → हट जाती है।

Example:
  openApps = ["terminal", "spotify", "notes"]
  User Notes का X दबाता है
  closeAppHandler("notes") call होता है
  openApps = ["terminal", "spotify"] ✅ सिर्फ Notes बंद हुई!

━━ Docs.jsx — Windows कैसे render होती हैं? ━━

FILE: src/components/Doc/Docs.jsx

  {openApps.includes("notes")    && <Notes    onClose={() => closeAppHandler("notes")}    />}
  {openApps.includes("folder")   && <Files    onClose={() => closeAppHandler("folder")}   />}
  {openApps.includes("terminal") && <Terminal onClose={() => closeAppHandler("terminal")} />}
  {openApps.includes("spotify")  && <Spotify  onClose={() => closeAppHandler("spotify")}  />}

यह "conditional rendering" है।
  openApps.includes("spotify") → check करता है: क्या "spotify" array में है?
  अगर हाँ  → <Spotify /> component render होता है (window दिखती है)
  अगर नहीं → कुछ नहीं दिखता

&& (AND operator) का मतलब:
  पहली condition true हो तो ही दूसरी चीज़ दिखाओ।

━━ Icon.js — Spotify का type क्यों बदला? ━━

FILE: src/utils/Icon.js

पहले था:  type: "iframe"
अब है:   type: "spotify"

कारण:
  Docs.jsx का handleOpenWindow function check करता है कि
  item.type क्या है।

  अगर type === "iframe" और url है → browser में नया tab खुलता है
  अगर type === "spotify"          → openAppHandler("spotify") call होता है
                                    और OS के अंदर window खुलती है

Spotify का url comment में था (कोई url नहीं था), इसलिए
"iframe" type से वो कहीं नहीं जाता था।
"spotify" type से वो सीधे OS की window में खुलता है।

================================================================
COMPLETE DATA FLOW — User click से Window तक का सफर
================================================================

STEP 1: User डेस्कटॉप पर Spotify icon click करता है
         ↓
STEP 2: Desktop.jsx → onClick → openWindow(item) call होता है
         item = { id:4, name:"Spotify", type:"spotify", ... }
         ↓
STEP 3: Docs.jsx → handleOpenWindow(item)
         item.type === "spotify" → openAppHandler("spotify") call
         ↓
STEP 4: App.jsx → openAppHandler("spotify")
         openApps array में "spotify" add होता है
         openApps = ["spotify"]
         ↓
STEP 5: React re-render होता है
         Docs.jsx को नया openApps मिलता है
         ↓
STEP 6: Docs.jsx check करता है: openApps.includes("spotify") → TRUE
         <Spotify onClose={...} /> render होता है
         ↓
STEP 7: Spotify.jsx render होता है
         <AppWindow> window screen पर आती है
         <iframe> Spotify की website load करता है
         ↓
STEP 8: Browser spotify embed URL fetch करता है
         Real-time Spotify player दिखता है ✅
